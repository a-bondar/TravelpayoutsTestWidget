Примеры использования виджета

Виджет размера XL с настройками по умолчанию


Виджет размера L. Пример инициализации: 
tpWidgetInit('widgetL', { textColor: 'black', backgroundColor: 'darkSeaGreen', buttonColor: 'darkCyan' })

tpWidgetInit('widgetM', { lang: 'ru', buttonColor: 'magenta' })

tpWidgetInit('widgetS', { lang: 'ru', buttonColor: '#48D1CC' })
