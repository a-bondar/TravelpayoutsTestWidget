Примеры использования виджета

Виджет с настройками по умолчанию


Виджет размера "xl". Пример инициализации: 
tpWidgetInit('widgetXL', { textColor: 'black', size: 'xl' })


Виджет размера "l". Пример инициализации: 
tpWidgetInit('widgetL', { textColor: 'black', backgroundColor: 'darkSeaGreen', buttonColor: 'darkCyan', size: 'l' })


Виджет размера "m". Пример инициализации: 
tpWidgetInit('widgetM', { lang: 'ru', buttonColor: 'magenta', size: 'm' })


Виджет размера "s". Пример инициализации: 
tpWidgetInit('widgetS', { lang: 'ru', buttonColor: '#48D1CC', size: 's' })
